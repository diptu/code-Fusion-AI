Code Fusion AI - Python Developer Assignment

May 4, 2025

1. Overview

This assignment involves fetching data from an external API, formatting the data, and making it available via RESTful APIs. The data will be displayed on a secured web page where users can view country information using Django.

2. Assignment Phases

2.1 Phase 1: Data Fetching and Storage
  • Start a new Django project and app.
  • Fetch data from REST Countries API.
  • Design and implement models to store data.
  • Write a script to fetch and store data in the database.
2.2 Phase 2: RESTful API Development
  • List all countries / Retrieve / Create / Update / Delete country data.
  • List regional or language-based countries.
  • Search by country name.
2.3 Phase 3: Web Interface
  • Show country table: name, cca2, capital, population, timezone, and flag.
  • Include search + details view (regional/language countries).
  • Styled with Bootstrap or Bulma.
2.4 Phase 4: Authentication and Security
  • Restrict API access to authenticated users.
  • Use Django's default authentication.
  • Login required to access country list page.

3. Deliverables

Push your code to a public Git repository with setup instructions and commit history.

4. Special Instructions

  • Use Python/Django for development.
  • Include third-party dependencies (in requirements.txt).
  • Provide a complete README.md with installation, dependencies, DB setup, and run instructions.
